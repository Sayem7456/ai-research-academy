<Callout type="info">Partial derivatives measure how a multivariable function changes with respect to one variable while holding others constant — essential for working with loss functions that depend on many parameters.</Callout>

## What Problem Does It Solve?

A loss function depends on thousands (or millions) of parameters. To optimize, we need to know how the loss changes with respect to each individual parameter. Partial derivatives provide this one-at-a-time analysis.

## Definition

For $f(x, y)$, the partial derivative with respect to $x$ is:

$$
\frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x + h, y) - f(x, y)}{h}
$$

Treat all other variables as constants.

## Notation

$$
\frac{\partial f}{\partial x} = f_x = \partial_x f = D_x f
$$

## Computation

For $f(x, y) = x^2 y + 3xy^2$:

$$
\frac{\partial f}{\partial x} = 2xy + 3y^2 \quad \text{(treat } y \text{ as constant)}
$$

$$
\frac{\partial f}{\partial y} = x^2 + 6xy \quad \text{(treat } x \text{ as constant)}
$$

## Mixed Partial Derivatives

$$
\frac{\partial^2 f}{\partial x \partial y} = \frac{\partial}{\partial x}\left(\frac{\partial f}{\partial y}\right)
$$

### Clairaut's Theorem

If the mixed partial derivatives are continuous, the order doesn't matter:

$$
\frac{\partial^2 f}{\partial x \partial y} = \frac{\partial^2 f}{\partial y \partial x}
$$

This symmetry is why the Hessian matrix is always symmetric.

<Callout type="tip">In PyTorch, `loss.backward()` computes all partial derivatives simultaneously using automatic differentiation — but understanding partial derivatives is still essential for debugging and designing custom losses.</Callout>

```python
import numpy as np

def f(x, y):
    return x**2 * y + 3 * x * y**2

def df_dx(x, y):
    return 2*x*y + 3*y**2

def df_dy(x, y):
    return x**2 + 6*x*y

# Verify numerically
x, y = 2.0, 3.0
h = 1e-7

# Partial w.r.t. x
num_dx = (f(x+h, y) - f(x-h, y)) / (2*h)
print(f"∂f/∂x analytical: {df_dx(x,y)}, numerical: {num_dx:.4f}")

# Partial w.r.t. y
num_dy = (f(x, y+h) - f(x, y-h)) / (2*h)
print(f"∂f/∂y analytical: {df_dy(x,y)}, numerical: {num_dy:.4f}")
```

## Key Takeaway

Partial derivatives let us analyze how a multi-parameter function changes along each dimension independently. They're the building blocks of gradients, Jacobians, and Hessians — all essential tools for optimization in machine learning.
