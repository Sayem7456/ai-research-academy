<Callout type="info">The chain rule computes derivatives of composite functions — it's the mathematical engine behind backpropagation in neural networks.</Callout>

## What Problem Does It Solve?

Neural networks are compositions of many functions: $\mathbf{y} = f_L(f_{L-1}(\cdots f_1(\mathbf{x})))$. How do we compute the derivative of the loss with respect to an early layer's weights? The chain rule tells us.

## Statement

If $y = f(g(x))$, then:

$$
\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} \quad \text{where } u = g(x)
$$

Or equivalently:

$$
\frac{d}{dx} f(g(x)) = f'(g(x)) \cdot g'(x)
$$

## Examples

### Example 1: $y = \sin(x^2)$

Let $u = x^2$, then $y = \sin(u)$:

$$
\frac{dy}{dx} = \cos(u) \cdot 2x = 2x \cos(x^2)
$$

### Example 2: $y = e^{3x+1}$

Let $u = 3x + 1$, then $y = e^u$:

$$
\frac{dy}{dx} = e^u \cdot 3 = 3e^{3x+1}
$$

### Multi-layer Composition

For $y = f(g(h(x)))$:

$$
\frac{dy}{dx} = f'(g(h(x))) \cdot g'(h(x)) \cdot h'(x)
$$

## Connection to Backpropagation

<Callout type="tip">Backpropagation IS the chain rule applied systematically. Each layer passes its local gradient backward, and the chain rule multiplies these local gradients together.</Callout>

Consider a 2-layer network: $\hat{y} = \sigma(W_2 \cdot \text{ReLU}(W_1 \mathbf{x}))$

The loss gradient with respect to $W_1$ requires the chain rule:

$$
\frac{\partial \mathcal{L}}{\partial W_1} = \frac{\partial \mathcal{L}}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z_2} \cdot \frac{\partial z_2}{\partial h} \cdot \frac{\partial h}{\partial z_1} \cdot \frac{\partial z_1}{\partial W_1}
$$

Each term is a **local gradient** — easy to compute at each layer.

## Computational Graphs

A computational graph makes the chain rule visual:

```
x → [×W₁] → z₁ → [ReLU] → h → [×W₂] → z₂ → [σ] → ŷ → [Loss] → L
```

Forward pass computes values; backward pass computes gradients by multiplying local gradients along the path.

```python
import numpy as np

# Manual backpropagation example
x = np.array([1.0, 2.0])
W1 = np.array([[0.5, -0.3], [0.2, 0.8]])
W2 = np.array([0.4, -0.6])

# Forward pass
z1 = W1 @ x           # Linear
h = np.maximum(0, z1)  # ReLU
z2 = W2 @ h            # Linear
y_hat = 1 / (1 + np.exp(-z2))  # Sigmoid

# Backward pass (chain rule!)
dL_dy = y_hat - 1.0   # Assume target = 1
dy_dz2 = y_hat * (1 - y_hat)  # Sigmoid derivative
dz2_dh = W2            # Linear derivative
dh_dz1 = (z1 > 0).astype(float)  # ReLU derivative

# Gradient for W1 (chain rule product)
dL_dW1 = np.outer(dL_dy * dy_dz2 * (dz2_dh * dh_dz1), x)
print(f"Gradient dL/dW1:\n{dL_dW1}")
```

## Key Takeaway

The chain rule is the single most important concept connecting calculus to deep learning. It enables efficient computation of gradients through composed functions — making training deep neural networks possible. Without the chain rule, backpropagation wouldn't exist.
