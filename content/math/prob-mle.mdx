<Callout type="info">MLE finds the parameters that make observed data most probable — the workhorse of classical statistics and the theoretical foundation of most loss functions in machine learning.</Callout>

## What Problem Does It Solve?

We have data and a model family. Which specific parameters should we choose? MLE says: pick the parameters that make the data most likely.

## Principle

$$
\hat{\theta}_{MLE} = \arg\max_\theta \prod_{i=1}^{n} p(x_i | \theta) = \arg\max_\theta \sum_{i=1}^{n} \log p(x_i | \theta)
$$

## Examples

### Gaussian MLE

For data $x_1, \ldots, x_n \sim N(\mu, \sigma^2)$:

$$
\hat{\mu} = \frac{1}{n} \sum_{i=1}^{n} x_i \quad \text{(sample mean)}
$$

$$
\hat{\sigma}^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \hat{\mu})^2 \quad \text{(sample variance)}
$$

### Linear Regression as MLE

Assume $y_i = \mathbf{w}^T \mathbf{x}_i + \epsilon_i$ where $\epsilon_i \sim N(0, \sigma^2)$. The MLE for $\mathbf{w}$ is:

$$
\hat{\mathbf{w}}_{MLE} = \arg\min_{\mathbf{w}} \sum_{i=1}^{n} (y_i - \mathbf{w}^T \mathbf{x}_i)^2
$$

This is exactly **ordinary least squares**!

<Callout type="tip">This is why MSE loss works — it's the negative log-likelihood under a Gaussian noise assumption. Every loss function has a probabilistic interpretation.</Callout>

### Classification as MLE

For binary classification with labels $y_i \in \{0, 1\}$:

$$
\hat{\mathbf{w}} = \arg\max_{\mathbf{w}} \sum_{i=1}^{n} \left[y_i \log \hat{y}_i + (1-y_i) \log(1-\hat{y}_i)\right]
$$

This is exactly **binary cross-entropy loss** — the negative log-likelihood of a Bernoulli model.

## Connection to Loss Functions

| Loss Function | Probabilistic Model |
|--------------|-------------------|
| MSE (L2) | Gaussian noise |
| MAE (L1) | Laplace noise |
| Cross-entropy | Categorical/Multinoulli |
| Hinge loss | Approximation to logistic |

## Properties of MLE

1. **Consistency**: $\hat{\theta}_{MLE} \to \theta^*$ as $n \to \infty$
2. **Efficiency**: achieves the Cramér-Rao lower bound asymptotically
3. **Invariance**: $g(\hat{\theta}_{MLE})$ is the MLE of $g(\theta)$

```python
import numpy as np

# MLE for Gaussian
data = np.array([2.3, 1.8, 3.1, 2.5, 2.9, 1.5, 2.7])
mu_mle = np.mean(data)
sigma2_mle = np.mean((data - mu_mle)**2)
print(f"MLE: μ = {mu_mle:.4f}, σ² = {sigma2_mle:.4f}")

# MLE for linear regression
np.random.seed(42)
X = np.random.randn(100, 2)
true_w = np.array([3.0, -1.5])
y = X @ true_w + 0.5 * np.random.randn(100)

# MLE solution: w = (X^T X)^{-1} X^T y
w_mle = np.linalg.inv(X.T @ X) @ X.T @ y
print(f"True weights: {true_w}")
print(f"MLE weights:  {w_mle.round(3)}")
```

## Key Takeaway

MLE is the theoretical foundation for almost all loss functions in machine learning. MSE loss is Gaussian MLE. Cross-entropy is Bernoulli/Categorical MLE. Understanding this connection explains why these losses work and how to design new ones for custom probabilistic models.
