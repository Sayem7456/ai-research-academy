<Callout type="info">Tensors are multi-dimensional arrays — the generalization of scalars, vectors, and matrices. They are the fundamental data structure in deep learning frameworks.</Callout>

## What Problem Does It Solve?

Real-world data isn't always flat. Images have height, width, and color channels. Videos add a time dimension. Tensors provide a unified way to represent data of any dimensionality.

## Definition

A **tensor** is a multi-dimensional array. The number of dimensions is called the **rank** (or order):

| Rank | Name | Example |
|------|------|---------|
| 0 | Scalar | $5$ |
| 1 | Vector | $[1, 2, 3]$ |
| 2 | Matrix | $\begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix}$ |
| 3 | 3-Tensor | Color image: $H \times W \times C$ |
| 4 | 4-Tensor | Batch of images: $N \times H \times W \times C$ |

$$
\mathcal{T} \in \mathbb{R}^{d_1 \times d_2 \times \cdots \times d_n}
$$

## Tensor Rank vs Matrix Rank

<Callout type="warning">Don't confuse tensor rank (number of dimensions) with matrix rank (number of linearly independent rows/columns). They're different concepts!</Callout>

## Tensor Operations

### Element-wise Operations

$$
(\mathcal{A} + \mathcal{B})_{i_1, i_2, \ldots, i_n} = a_{i_1, i_2, \ldots, i_n} + b_{i_1, i_2, \ldots, i_n}
$$

### Tensor Contraction

Generalizes matrix multiplication to higher dimensions. Sum over shared indices:

$$
C_{ik} = \sum_j A_{ij} B_{jk} \quad \text{(matrix multiplication as tensor contraction)}
$$

### Broadcasting

Broadcasting allows operations on tensors of different shapes by automatically expanding dimensions:

$$
\text{Shape } (3, 1) + \text{Shape } (1, 4) \to \text{Shape } (3, 4)
$$

## Tensors in Deep Learning

```python
import torch

# Scalar (rank 0)
s = torch.tensor(5.0)

# Vector (rank 1)
v = torch.tensor([1.0, 2.0, 3.0])

# Matrix (rank 2)
M = torch.tensor([[1, 2], [3, 4]])

# 3D Tensor — single RGB image (C, H, W)
img = torch.randn(3, 224, 224)

# 4D Tensor — batch of images (N, C, H, W)
batch = torch.randn(32, 3, 224, 224)

# Tensor operations
print(batch.shape)        # torch.Size([32, 3, 224, 224])
print(batch.ndim)          # 4
print(batch.numel())       # 32 * 3 * 224 * 224 = 4,816,896

# Reshaping
flat = batch.view(32, -1)  # (32, 150528)

# Broadcasting
bias = torch.randn(3, 1, 1)
result = img + bias  # Broadcasting: (3,224,224) + (3,1,1) → (3,224,224)
```

## Key Takeaway

Every piece of data in a deep learning framework is a tensor. Images, text embeddings, audio signals, video frames — all represented as multi-dimensional arrays. Understanding tensor shapes, reshaping, and broadcasting is essential for building and debugging neural networks.
