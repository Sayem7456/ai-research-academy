<Callout type="info">MAP estimation incorporates prior beliefs into parameter estimation — bridging frequentist MLE and full Bayesian inference. It's equivalent to adding regularization to loss functions.</Callout>

## What Problem Does It Solve?

MLE can overfit when data is scarce — it trusts only the data. MAP adds prior knowledge as a regularization term, producing more robust estimates especially with limited data.

## Principle

$$
\hat{\theta}_{MAP} = \arg\max_\theta P(\theta | \mathcal{D}) = \arg\max_\theta \frac{P(\mathcal{D}|\theta) P(\theta)}{P(\mathcal{D})}
$$

Since $P(\mathcal{D})$ doesn't depend on $\theta$:

$$
\hat{\theta}_{MAP} = \arg\max_\theta P(\mathcal{D}|\theta) P(\theta) = \arg\max_\theta \left[\log P(\mathcal{D}|\theta) + \log P(\theta)\right]
$$

## MAP = MLE + Prior

$$
\hat{\theta}_{MAP} = \arg\max_\theta \underbrace{\log P(\mathcal{D}|\theta)}_{\text{log-likelihood (MLE)}} + \underbrace{\log P(\theta)}_{\text{log-prior (regularization)}}
$$

The prior acts as a **regularizer** that pulls estimates toward preferred values.

## Regularization as MAP

<Callout type="tip">This is one of the most important connections in machine learning: common regularization techniques correspond to specific prior distributions.</Callout>

| Regularization | Prior | Effect |
|---------------|-------|--------|
| **L2** (Ridge) $\lambda\|\mathbf{w}\|_2^2$ | Gaussian: $\mathbf{w} \sim N(0, \sigma^2 I)$ | Shrinks weights toward zero |
| **L1** (Lasso) $\lambda\|\mathbf{w}\|_1$ | Laplace: $w_i \sim \text{Laplace}(0, b)$ | Drives weights to exactly zero (sparsity) |
| **Elastic Net** | Combination | Both shrinkage and sparsity |

### Why L2 = Gaussian Prior

Gaussian prior: $P(\mathbf{w}) \propto \exp\left(-\frac{\|\mathbf{w}\|_2^2}{2\sigma^2}\right)$

Log-prior: $\log P(\mathbf{w}) = -\frac{1}{2\sigma^2}\|\mathbf{w}\|_2^2 + \text{const}$

This is exactly the L2 regularization term with $\lambda = \frac{1}{2\sigma^2}$.

### Why L1 = Laplace Prior

Laplace prior: $P(w) \propto \exp\left(-\frac{|w|}{b}\right)$

Log-prior: $\log P(w) = -\frac{|w|}{b} + \text{const}$

This is the L1 regularization term. The Laplace distribution has a sharp peak at zero, promoting sparsity.

## Comparison: MLE vs MAP

| Aspect | MLE | MAP |
|--------|-----|-----|
| Uses prior | No | Yes |
| Regularization | None | Implicit via prior |
| With lots of data | Same as MAP | Same as MLE |
| With little data | Can overfit | More robust |
| Computation | Optimization | Optimization |

<Callout type="info">As $n \to \infty$, the likelihood dominates the prior, and MAP converges to MLE. The prior matters most when data is scarce.</Callout>

```python
import numpy as np

# MAP for linear regression with L2 regularization (Ridge)
np.random.seed(42)
X = np.random.randn(20, 5)
true_w = np.array([1.0, -0.5, 0, 0, 0.3])
y = X @ true_w + 0.5 * np.random.randn(20)

# MLE (no regularization)
w_mle = np.linalg.inv(X.T @ X) @ X.T @ y

# MAP with Gaussian prior (Ridge regression)
lam = 1.0  # regularization strength = 1/σ²_prior
w_map = np.linalg.inv(X.T @ X + lam * np.eye(5)) @ X.T @ y

print("True weights:  ", true_w.round(3))
print("MLE weights:   ", w_mle.round(3))
print("MAP (Ridge):   ", w_map.round(3))
print(f"MLE ||w||: {np.linalg.norm(w_mle):.3f}")
print(f"MAP ||w||: {np.linalg.norm(w_map):.3f}")  # Smaller!

# MAP with Laplace prior (Lasso) — coordinate descent
from scipy.optimize import minimize
def lasso_objective(w):
    residuals = y - X @ w
    mse = np.sum(residuals**2)
    l1 = lam * np.sum(np.abs(w))
    return mse + l1

result = minimize(lasso_objective, np.zeros(5), method='Nelder-Mead')
w_lasso = result.x
print("MAP (Lasso):   ", w_lasso.round(3))  # More sparse!
```

## Key Takeaway

MAP estimation elegantly unifies MLE and regularization. L2 regularization is a Gaussian prior; L1 is a Laplace prior. This probabilistic interpretation helps choose appropriate regularization, design custom priors for domain knowledge, and understand the fundamental tradeoff between data fit and prior beliefs.
